Înainte de a instala un editor/IDE, avem nevoie de un compilator. Există următoarele opțiuni:

<Cards>
  <Card title="MSVC" href="./editors/windows/visual-studio-2022.mdx">
    MSVC (Microsoft Visual C++) este un compilator proprietar făcut de Microsoft și este parte din IDE-ul Visual Studio 2022.

    Dacă ești sigur(ă) că vrei să folosești acest IDE în particular, alege această opțiune. Dacă l-ai instalat deja, atunci alege această opțiune, deoarece articolul include și niște sfaturi de performanță.
  </Card>

  <Card title="GCC (MinGW)" href="./compilers/windows/mingw64.mdx">
    GCC (GNU Compiler Collection) este o suită de compilatoare libere și gratuite. Cel mai folosit compilator din această suită este GCC (GNU C Compiler).

    Pe Windows, cea mai ușoară metodă de a instala compilatorul GCC este folosind MinGW-w64. În caz că nu folosești Visual Studio 2022, ci alt editor (de pildă, Code::Blocks sau Visual Studio Code), alege această opțiune.
  </Card>
</Cards>
